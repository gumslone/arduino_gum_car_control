# arduino_gum_car_control
android bluetooth controller for arduino car

button - bluetooth command:
- up - F
- down - B
- left - L
- right - R
- up+right - I
- up+left - G
- down+right - J
- down+left - H

Lights on: W

Lights off: w

Sound on: V

Sound off: v

Auto mode (obstacle avoidance):
-Mode on: X
-Mode off: x

Default:
-stop - S
